# 第四章 实验结果与分析

> 注：本章当前完成 `4.1`、`4.2` 的结构化草稿与表格骨架。待实际跑通基线或确定最终引用方式后，再填入精确硬件参数、超参数与实验结果。

## 4.1 实验环境与数据集准备

### 4.1.1 实验环境

为验证本文所提方法在低光照图像增强任务中的有效性，实验部分将在统一的深度学习环境中完成。考虑到本文方法涉及 Retinex 近似分解、MLLM 离线提示生成以及扩散模型推理等多个模块，整体实验流程采用“离线先验生成 + 在线扩散恢复”的两阶段实现方式。

在软件环境方面，本文主要基于 PyTorch 框架实现，并结合 diffusers、transformers 与 BasicSR 等工具链完成扩散恢复、特征编码和部分基线模型复现。由于不同对比方法的官方实现环境存在差异，例如 URetinex-Net 依赖较早版本的 PyTorch，SSP-IR 和 SeeSR 基于 Stable Diffusion 生态，而 Reti-Diff 与 Diff-Retinex 则具有各自独立的配置要求，因此本文在复现实验中优先遵循官方仓库提供的依赖说明，以降低因环境差异造成的结果偏差。

在硬件配置方面，正式定稿时需补充具体 GPU 型号、显存大小、CPU 与运行时长统计。对于本文方法及相关扩散基线，实验中重点记录以下三类效率指标：单张图像平均推理时间、显存占用以及扩散采样步数。该设置有助于更全面地评价本文方法在恢复性能与实际可部署性之间的平衡关系。

为保证实验比较的可复现性，本文对所有涉及随机性的实验统一设置随机种子，并在 paired 数据集上优先采用官方测试脚本进行评估；对于需要额外重采样、分辨率缩放或图像裁剪的设置，将在具体结果表格中单独注明。

### 4.1.2 数据集准备与划分

结合公开代码仓库与现有文献中的常用测试协议，本文将实验数据划分为主量化数据集与辅助真实场景展示数据集两部分。

主量化实验采用以下三组成对数据集：

1. LOL-v1：该数据集是低光照图像增强中最常用的 paired 数据集之一，广泛用于 URetinex-Net、Diff-Retinex 等方法的定量评估。
2. LOLv2-Real：该数据集包含真实低光照采集样本，能够更好反映模型在真实成像噪声与曝光失衡条件下的恢复能力。
3. LOLv2-Synthetic：该数据集通过合成退化方式构造低光样本，适合评估模型在可控退化条件下的稳定性与重建上限。

辅助真实场景展示实验可采用 LIME、NPE、DICM、MEF 和 VV 等公开无参考低光照数据集，用于展示模型在复杂真实场景中的视觉增强效果。如果后续时间允许，也可进一步引入 RealLR200 或 MIT-Adobe FiveK 中的低光子集，用于补充真实场景的多样化展示。

在预处理策略上，本文优先遵循各基线方法的官方测试流程，以保证比较公平性。对于扩散模型相关方法，若官方实现采用固定处理尺度，则在对应实验中保留其默认设置。例如，SSP-IR 公布的推理脚本默认采用 `process_size = 512` 与 `50` 步扩散采样；Reti-Diff 则区分 `LLIE_real` 和 `LLIE_syn` 两套测试配置。本文方法在与其对比时，将尽量保持相同或可对照的推理预算。

值得注意的是，部分方法在测试阶段使用特殊的亮度重标定策略。例如，Diff-Retinex 官方代码中提供了 `GT-means` 选项，该选项会利用真实目标图像的均值信息重新调整预测结果亮度，可能显著影响 PSNR 指标。为保证比较公平，本文在后续实验中默认关闭此类依赖目标图像统计信息的后处理，并在表格中明确说明相关设置。

## 4.2 评估指标与对比算法

### 4.2.1 评估指标

针对不同类型的数据集与评价目标，本文采用有参考指标、无参考指标和效率指标相结合的方式，对增强结果进行综合分析。

在成对数据集上，本文主要采用峰值信噪比（PSNR）、结构相似性指数（SSIM）以及感知相似性指标（LPIPS）进行评价。其中，PSNR 与 SSIM 用于衡量增强结果与目标图像之间的像素级重建保真度和结构一致性；LPIPS 则能够从感知特征空间衡量输出图像与参考图像之间的视觉接近程度，从而弥补单纯像素指标无法充分体现感知质量的问题。

在真实场景无参考数据集上，本文优先考虑自然图像质量评价指标 NIQE，并视后续实验条件补充 LOE 等低光照相关无参考指标。相较于 FID，NIQE 在小规模低光照测试集上更易于部署和解释，而 FID 对样本量、特征提取实现和统计稳定性要求较高，因此本文暂不将 FID 作为核心主指标。如后续实验引入 FID，则必须在正文中单独说明其实现细节、统计样本规模和评估协议。

此外，考虑到本文方法基于冻结扩散主干与多先验引导机制构建，其计算复杂度高于普通判别式增强模型，因此实验中还将补充单张图像平均推理时间、显存占用和扩散步数等效率指标，用于分析本文方法在性能收益与计算代价之间的关系。

### 4.2.2 对比算法选择原则

为了全面评估本文方法的性能，本文从传统对抗式方法、Retinex 判别式方法、Retinex 与扩散结合方法以及语义/先验引导扩散方法四类模型中选取代表性算法进行比较。

第一类为 GAN 或早期生成式增强方法，以 EnlightenGAN 为代表。该类方法能够在一定程度上改善图像视觉锐度，是低光照增强中常见的经典感知型基线。

第二类为基于 Retinex 理论的判别式增强方法，包括 URetinex-Net 和 ILR-Net。该类方法具有较好的物理解释性，适合作为本文前置 Retinex 分解思想的直接对照组。

第三类为 Retinex 与扩散模型结合的方法，包括 Diff-Retinex 与 Reti-Diff。此类方法与本文在研究主题上更为接近，是评估“Retinex + 生成模型”路线有效性的关键比较对象。

第四类为语义或结构先验引导的扩散恢复方法，以 SSP-IR 为代表。该类方法与本文的总体设计思路最为接近，因此也是最重要的主对比对象之一。

在结果来源方面，本文优先使用官方公开代码与官方预训练权重进行复现。当某一方法因环境、依赖或模型权重限制暂时无法完整复现时，可在明确注明来源和测试协议的前提下引用论文报告结果，但必须在正文中标注为“论文报告结果”而非“本文复现结果”，避免混淆。
