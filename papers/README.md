# Papers Folder

Store papers related to the thesis here, together with short summaries when needed.

## Recommended Categories

- Retinex theory and decomposition
- classical low-light enhancement methods
- deep learning low-light enhancement methods
- papers directly related to `sspir`
- datasets and evaluation metrics for low-light enhancement

## Recommended Naming

Use a stable filename pattern such as:

`year_firstauthor_keyword.pdf`

Examples:

- `2011_jobson_retinex.pdf`
- `2023_zhang_lowlight_transformer.pdf`

## Suggested Companion Notes

For important papers, add a note file with the same stem, for example:

- `2023_zhang_lowlight_transformer.pdf`
- `2023_zhang_lowlight_transformer.md`

Each note can briefly record:

- problem addressed
- core method
- dataset and metrics
- strengths
- weaknesses
- what is useful for this thesis
